# 💱 Currency Converter

A minimal and modern currency converter built with **HTML, CSS, and JavaScript**.  
It fetches real-time exchange rates using the **ExchangeRate API**, with smooth UI transitions and a clean layout.

---

## ✅ Features
✔ Real-time currency conversion  
✔ 100+ currencies supported  
✔ Swap button (invert currencies instantly)  
✔ Minimal and responsive UI  

---

## 🛠 Tech Used
- HTML
- CSS
- JavaScript (Fetch API)
- ExchangeRate API

---

## 📦 How It Works
1. Select the source currency  
2. Select the target currency  
3. Enter an amount  
4. Click **Convert**  
5. The result appears with animation and formatted value
